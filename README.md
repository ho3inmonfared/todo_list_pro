# Todo List Pro

![Python](https://img.shields.io/badge/python-3.11-blue) 
![Django](https://img.shields.io/badge/django-4.3-green) 
![License](https://img.shields.io/badge/license-MIT-blue)

**Todo List Pro** is a modern, responsive **task management application** built with **Django** and **Tailwind CSS**. It is designed for managing daily tasks, work projects, and personal activities. This project allows users to **add, edit, and delete tasks**, categorize them, and track their completion status.

---

## Features

- Add new tasks with a specific category
- Display completed and pending tasks
- Track task completion status (`is_complete`)
- Modern UI design using **Tailwind CSS** and glassmorphism effects
- Responsive and mobile-friendly interface
- Color-coded categories for better visualization
- Secure forms with CSRF protection and server-side validation
- Easily extensible for advanced features like priorities, reminders, and APIs
